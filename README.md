# Задание 1

## Задача 1

### Текст задачи

Сформировать список из чисел, обратных вводимым значениям.

### Алгоритм решения

первое число n - число элементов списка. Далее запрашиваем у пользователя n раз число a, сохраняем каждое новое a как 1/a, т.е. обратное число

выводим пользователю список из чисел, обратных введённым

### Тестирование

Скриншоты результата работы программы

## Задача 2

### Текст задачи

Найти количество чётных цифр в записи натурального числа.

### Алгоритм решения

пользователь последовательно вводит числа (до тех пор, пока не захочет выйти, для этого ввести q). У каждого введённого числа рекурсивно перебираем все цифры и увеличиваем счётчик, если цифра - чётная. Выводим получившуюся сумму пользователю

### Тестирование

Скриншоты результата работы программы



# Задание 2

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными

числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать

готовый класс Complex запрещено.

### Алгоритм решения

Представим комплексное число как список из 2 элементов типа float - действительная часть и мнимая часть. Далее все операции происходят со списками.
пользователь вводит комплексное число - отдельно мнимая и действительная части. Допускается как наличие i в вводимой мнимой части, так и его отсутствие. Далее по тому же принципу вводится второе число. Далее, выбор операции (+-\*/^).

Производится операция, результат выводится на экран.

Повторяется до ввода пользователем q

### Тестирование

Скриншоты результата работы программы
